# BizTrip Budget Guard — Step-by-Step Setup Guide (Mac M2)

This guide assumes macOS on an Apple Silicon MacBook Air M2 with 8GB RAM.

---

## 1. Create Project Folder

Open **Terminal** and run:

```bash
cd ~/Desktop
mkdir -p biztrip-budget-guard
```

Inside that folder, create these exact subfolders:

```bash
cd biztrip-budget-guard
mkdir -p backend/services frontend/src/components frontend/src/services frontend/src/data
```

Paste every code file exactly into the paths shown in this project.

---

## 2. Create and Activate Python Virtual Environment

```bash
cd backend
python3 -m venv venv
source venv/bin/activate
```

Your prompt should now show `(venv)` at the front.

---

## 3. Install Backend Dependencies

With `venv` active, run:

```bash
pip install --upgrade pip
pip install -r requirements.txt
```

---

## 4. Create and Populate `.env` File

```bash
cp .env.example .env
```

Open `.env` in any text editor (e.g., `nano .env` or VS Code) and replace the placeholder keys.

### How to get free API keys

| Service | URL | Steps |
|---------|-----|-------|
| **Groq** | https://console.groq.com | Sign up free → Create API key → copy it |
| **Amadeus** | https://developers.amadeus.com | Register free → Create app (test environment) → copy Key + Secret |
| **ExchangeRate-API** | https://www.exchangerate-api.com | Sign up free → copy API key |

Paste those values into `.env`. Example:

```env
GROQ_API_KEY=gsk_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
AMADEUS_API_KEY=YOUR_AMADEUS_KEY
AMADEUS_API_SECRET=YOUR_AMADEUS_SECRET
EXCHANGE_RATE_API_KEY=YOUR_EXCHANGE_RATE_KEY
```

Save and close.

---

## 5. Set Up React Frontend with Vite

In a **new Terminal tab/window**:

```bash
cd ~/Desktop/biztrip-budget-guard/frontend
```

Scaffold the Vite project (choose **React** and **JavaScript** when prompted):

```bash
npm create vite@latest . -- --template react
```

If it asks to overwrite existing files, choose **yes** for `package.json`, `vite.config.js`, and `index.html` only if needed. Otherwise, keep the ones already provided.

Install frontend dependencies:

```bash
npm install
npm install react-router-dom axios recharts
```

---

## 6. Run Backend Server

In the **backend Terminal tab** (with `venv` active):

```bash
cd ~/Desktop/biztrip-budget-guard/backend
source venv/bin/activate
python main.py
```

The backend runs on **http://localhost:8000**

You should see:

```
INFO:     Uvicorn running on http://0.0.0.0:8000
```

Keep this tab running.

---

## 7. Run Frontend Dev Server

In the **frontend Terminal tab**:

```bash
cd ~/Desktop/biztrip-budget-guard/frontend
npm run dev
```

The frontend runs on **http://localhost:5173**

You should see:

```
  VITE v5.x.x  ready in xxx ms
  ➜  Local:   http://localhost:5173/
```

Keep this tab running.

---

## 8. Test Both Together in Browser

1. Open **Chrome / Safari / Firefox**
2. Go to **http://localhost:5173**
3. You should see the **BizTrip Budget Guard** nav bar
4. Try the **Forecast** tab, fill in a destination (e.g., `LON`), dates, and click **Generate Forecast**
5. Check the **Dashboard**, **Spend Tracker**, and **Reconcile** tabs after creating a trip

---

## Troubleshooting

- **Port already in use?** Change `PORT` in `.env` or run frontend with `npm run dev -- --port 3000`
- **CORS errors?** Make sure backend is running and `allow_origins=["*"]` is set in `main.py`
- **Module not found?** Ensure `venv` is activated before running backend
- **Groq rate limit?** The app automatically falls back to rule-based templates
